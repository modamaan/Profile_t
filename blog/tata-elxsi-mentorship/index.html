Get started with the MERN stack

#Engineering

MERN is a full-stack solution named after the technologies that make up the stack: MongoDB, Express, React, and Node.js.

MongoDB

MongoDB is a NoSQL document-based database. Databases are used to persist any data the users will need. In this guide, we are going to use MongoDB Atlas, MongoDB's managed database solution.

Express

Express.js is a flexible and minimalist web framework for building Node.js applications

React

React.js is a front-end frameowrk that lets you build interactive UIs.

Node.js

Node.js is an asynchronous event-driven JavaScript runtime designed to build scalable network applications.
